安装MySQL

我们去官网)去下载.dmg文件自己安装，这样可以在系统偏好设置中找到 MySQL 直接进行开关。
完安装完成后，开启数据库会给你一个默认的密码，记下密码作为以后登录的凭据

修改 mysql 密码
苹果->系统偏好设置->最下边点 mysql 在弹出页面中 关闭 mysql 服务（点击stop mysql server）

进入终端输入：cd /usr/local/mysql/bin/
回车后 登录管理员权限 sudo su
回车后输入以下命令来禁止mysql验证功能 ./mysqld_safe --skip-grant-tables &
回车后mysql会自动重启（偏好设置中 mysql 的状态会变成 running）

输入命令 ./mysql
回车后，输入命令 FLUSH PRIVILEGES
回车后，输入命令 SET PASSWORD FOR 'root'@'localhost' = PASSWORD('你的新密码')
至此，密码修改完成，可以成功登陆
